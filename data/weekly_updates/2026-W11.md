# 2026-W11（2026-03-02～2026-03-08）

## Weekly Summary

本周 TRAE 强化 Markdown preview、Agent terminal 展示和 terminal cards。看似是 UI 细节，本质是把 agent 执行过程变得更可读、更不吓人。

## Product Signals

- TRAE v3.3.35 增加 editor Markdown preview。
- Agent command terminal 不再默认打开原始 terminal，而是通过 terminal cards 等方式降低执行噪声。

## Product Interpretation

- **路线判断**：AI coding 产品要让用户信任 agent，不能只展示 raw shell。产品要把工具调用转译为用户能理解的状态、风险和结果。
- **对 TRAE SOLO 的启示**：面向非工程师，命令执行应被抽象成“读取数据 / 安装依赖 / 运行校验 / 生成预览 / 发布草稿”等业务动作，并保留可展开的技术细节。

## LPME Implication

- LPME 应记录“执行可解释性”：用户是否能在不懂 terminal 的情况下判断 agent 是否在做合理事情。
