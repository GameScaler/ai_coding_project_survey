# AI Coding Daily Update - 2026-06-09

## Summary

今天有 4 个产品级信号：Codex CLI 增加桌面端接力路径，Claude Code 强化排障与治理开关，TRAE SOLO 升级为 TRAE Work 并补齐企业账号入口，OpenClaw 继续强化多渠道 agent 输出与 MCP 工具结果处理。

## Product Updates

### OpenAI Codex
- Codex CLI 0.138.0 增加 `/app`，可把当前 CLI 线程接力到 macOS / Windows 的 Codex Desktop；结合模型文档中推荐模型与弃用模型边界更新，Codex 正在把“终端 agent”与“桌面任务工作台”的连续性做成默认路径。

### Claude Code
- Claude Code 2.1.169 增加 `--safe-mode` / `CLAUDE_CODE_SAFE_MODE`、`/cd` 会话换目录和禁用内置 skills 的配置，并修复企业托管 MCP policy 重连/首次会话不生效等问题；重点是让长会话排障、目录切换和组织级工具治理更可控。
- Claude Code Agent SDK 明确从 2026-06-15 起，订阅计划中的 Agent SDK 与 `claude -p` 用量将使用独立的月度 Agent SDK credit；SDK 正在从“CLI 能力包装”变成需要单独预算和用量治理的生产化接口。
