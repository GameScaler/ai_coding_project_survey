# AI Coding Daily Update - 2026-06-01

## Summary

重大更新初筛：有中等重要更新。

TRAE SOLO 桌面端继续强化浏览器上下文：支持在内置浏览器选中元素并添加到对话/评论。这是 More Than Coding 里非常关键的交互入口。

## Product Updates

### TRAE SOLO

- TRAE 中国 changelog 显示 2026-06-01 v0.1.13-0.1.15 中，SOLO 桌面端支持在内置浏览器选中元素并添加到对话/评论。
- 该能力强化了“视觉/页面上下文 -> 对话 -> 修改/评论”的闭环。

### Others

- OpenAI Codex、Claude Code、Cursor、GitHub Copilot、Windsurf/Devin 未观察到同等级别重大公开更新。

## PM Notes

- **产品官短评**：这类功能看似很小，但对非工程师非常重要。用户不一定知道 DOM、组件、文件路径，但可以指着页面说“这个地方不对”。把视觉选择转成 agent context，是 Work 模式走向多角色的关键。
- **对 TRAE SOLO 的启示**：应把“圈选/评论/截图”做成一等上下文，并和验证闭环打通：用户圈选问题，agent 修改，浏览器验证，用户再确认。
- **LPME 是否更新**：建议在前端/营销/PM 任务中加入视觉批注验收步骤。
