# Feature Brief: PM Review Board

Build a lightweight review board for product managers who use AI agents to create project artifacts.

## User Need

PMs receive many AI-generated artifacts: PRDs, landing pages, dashboards, analysis memos. They need one place to see what is ready, what needs review, and what has unresolved risk.

## Must Have

- Task list grouped by status.
- Artifact preview area.
- Risk checklist.
- Reviewer comments.
- "Mark ready" action.

## Non-goals

- No real authentication.
- No backend database.
- No external deployment.

## Success Metrics

- Time to first review under 3 minutes.
- PM can identify unresolved risk without reading code.
- Engineer receives clear acceptance checklist.
